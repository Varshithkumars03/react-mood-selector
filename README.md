# React Mood Selector

React Mood Selector is an interactive web application that allows users to select their current mood from different options such as Happy, Neutral, Sad, and Angry.

The application dynamically updates the interface by displaying relevant emojis, messages, and background color changes based on the selected mood.

🌐 **Live Website**
https://varshithkumars03.github.io/react-mood-selector

---

## Features

* Select mood from multiple options
* Dynamic background color updates
* Emoji and message feedback
* Interactive and responsive UI
* Built using React state management

---

## Technologies Used

* React
* JavaScript (ES6)
* HTML5
* CSS3

---

## How to Run the Project

Clone the repository

```
git clone https://github.com/Varshithkumars03/react-mood-selector.git
```

Install dependencies

```
npm install
```

Run the project

```
npm start
```

The application runs at:

```
http://localhost:3000
```
